DECISIONS MADE
--------------
- GPG private key stays off Pi. TLS CA key on Pi is acceptable and unrelated.
- DB backups are also GPG encrypted (.db.gpg) — no plain .db files stored.
  "Restore directly" means download → decrypt → upload, not one-click. Modal
  in the Backup Manager explains this clearly.
- Config Editor new-keys modal fires for ALL pending notifications, not just
  required/placeholder keys. Integer defaults are still useful to surface.
- Retention config goes in Config Editor (config keys), not in the Backup
  Manager UI directly.
