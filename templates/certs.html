Install CA Certificate on Your Devices

Download and install the root CA certificate once on each device. After installation, NetWatch will show a clean padlock with no warnings.

Download netwatch-ca.crt
🖥 Windows
  1. Download the certificate above
  2. Double-click netwatch-ca.crt
  3. Click Install Certificate
  4. Choose Local Machine → Next
  5. Select Trusted Root Certification Authorities
  6. Finish → Yes
  7. Restart Chrome/Edge
📱 iPhone / iPad
  1. Download the certificate above in Safari
  2. Go to Settings → General
  3. Tap VPN & Device Management
  4. Tap the NetWatch CA profile → Install
  5. Go to Settings → General → About
  6. Tap Certificate Trust Settings
  7. Toggle NetWatch CA to ON
🤖 Android
  1. Download the certificate above
  2. Go to Settings → Security
  3. Tap Install from storage (or Encryption & credentials)
  4. Select netwatch-ca.crt
  5. Name it NetWatch CA
  6. Select CA Certificate
  7. Tap OK
🍎 macOS
  1. Download the certificate above
  2. Double-click netwatch-ca.crt
  3. Keychain Access opens — add to System keychain
  4. Find NetWatch Root CA in the list
  5. Double-click it → expand Trust
  6. Set When using this certificate to Always Trust
  7. Close and enter your Mac password
CA Backup

Download an encrypted backup of your CA private key. Store this somewhere safe — you'll need it if the Pi ever needs to be rebuilt. The backup is password-protected with the password you chose during setup.

Regenerate Server Certificate

If you add a new IP address to the Pi or the certificate expires, regenerate it here. This does NOT change the CA — your devices will not need to reinstall anything. NetWatch will restart briefly.
